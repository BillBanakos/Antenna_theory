# 📡 Antenna Theory Studio (with the help of Claude opus 4.8)

Διαδραστικό εκπαιδευτικό GUI (PyQt5 + matplotlib) που **οπτικοποιεί όλη τη θεωρία**
του βοηθήματος *«Θεωρία Κεραιών — Πλήρες Βοήθημα»* (βασισμένο στο **Balanis**,
Κεφ. 1, 2, 4, 6, 14). Κάθε έννοια γίνεται ζωντανό, παραμετρικό διάγραμμα — και
ένας ειδικός **Οπτικοποιητής Εννοιών Πολλαπλής Επιλογής** υλοποιεί ως σχήματα τις
«καθοριστικές διακρίσεις & παγίδες» κάθε κεφαλαίου.

---

## 🚀 Εκκίνηση

```bash
# 1) Εγκατάσταση εξαρτήσεων
pip install -r requirements.txt

# 2) Εκτέλεση (οποιοδήποτε από τα δύο)
python run.py
#   ή
python -m antenna_studio.main
```

> Απαιτείται γραφικό περιβάλλον (X server / Wayland / Windows / macOS) — το GUI
> χρησιμοποιεί το backend **Qt5Agg** της matplotlib.

## 🧭 Περιεχόμενα (πλοήγηση ανά κεφάλαιο)

| Κεφ. | Σελίδα | Τι δείχνει |
|------|--------|------------|
| **1** | Μηχανισμός & ρεύμα | Κινούμενη κατανομή ρεύματος `I(z')=I₀·sin[k(l/2−|z'|)]`, δεσμίδες φορτίου λ/2 |
| **2** | Διάγραμμα ακτινοβολίας | 3D επιφάνεια + πολική τομή (απειροστό/λ-2/πεπερασμένο/cosⁿθ/sin³θ), HPBW·FNBW·D₀ |
| **2** | Πεδιακές περιοχές | Αντιδραστική → Fresnel → Fraunhofer, όρια από D, λ, σφάλμα φάσης δ |
| **2** | Κατευθυντικότητα | Ακριβής vs Kraus/Tai–Pereira (μολύβι) & McDonald/Pozar (παντοκατευθυντικό) |
| **2** | Πόλωση (ζωντανή) | Έλλειψη πόλωσης + κινούμενο E + 3D έλικα· AR, τ, φορά, ταξινόμηση |
| **2** | PLF | Ζωντανή ζεύξη Tx→Rx, `PLF=|ρ̂_w·ρ̂_a|²` σε dB |
| **2** | Friis & Ραντάρ | `Pr/Pt` vs απόσταση, 1/R² (Friis) vs 1/R⁴ (ραντάρ/RCS) |
| **4** | Δίπολο & αντίσταση | Πολικό διάγραμμα + Rr/Rin vs μήκος + VSWR σε Z₀ |
| **4** | Θεωρία ειδώλων | AF στοιχείου+ειδώλου (PEC κατακόρυφο +1 / οριζόντιο −1), μηδενισμοί |
| **6** | Ομοιόμορφη — Beamsteering | N στοιχεία, β χειροκίνητο/broadside/end-fire/Hansen–Woodyard, SLL, D₀ |
| **6** | Σύνθεση | Ομοιόμορφη vs Διωνυμική vs Dolph–Tschebyscheff (στάθμη πλευρικών, πλάτη) |
| **6** | Επίπεδη M×N | Διαχωρίσιμο `AF=AFₓ·AF_y`, σάρωση (θ₀,φ₀), grating lobes (d≥λ) |
| **14** | Σχεδιαστής patch | Από fr/εr/h → W, L, εreff, ΔL, Rin· γεωμετρία + καμπύλη inset feed |
| **14** | Κυκλικό patch | Ρυθμοί TM110/TM210, ενεργός ακτίνα, κάτοψη πεδίου Ez |
| **MCQ** | Οπτικοποιητής εννοιών | Γκαλερί με τα κανονικά σχήματα & παγίδες όλων των κεφαλαίων |

## ✅ Επαλήθευση τύπων

Όλες οι συναρτήσεις του `core/formulas.py` έχουν ελεγχθεί αριθμητικά ως προς τις
λυμένες απαντήσεις του βοηθήματος, π.χ.:

- Δίπολο: `Rin(λ/4)≈13.7 Ω`, `Rin(λ/2)=73 Ω`, `Rin(3λ/4)≈372 Ω`
- `D₀(sin³θ)=16/3π=1.6977` (2.30 dB)
- Dolph N=5, −30 dB: `x₀=1.587`, συντελεστές `[1, 2.41, 3.14, 2.41, 1]`, `f=1.144`
- Patch 2 GHz, εr=10.2: `W≈3.17 cm`, `εreff≈9.38`, `L≈2.34 cm`
- Κυκλικό TM210 (1.9 GHz): `a≈2.40 cm`, `aₑ≈2.43 cm`
- Διωνυμικοί N=5: `[1, 4, 6, 4, 1]`
